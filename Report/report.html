Introduction

FORG (Food Organizer) is a web application which provides tools for managing culinary preferences, creating shopping lists and helping you discover recipes.

Used technologies

  1. HTML5

    2. HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. HTML5 is the latest evolution of the standard that defines HTML. It is a new version of the language HTML, with new elements, attributes, and behaviors, and a larger set of technologies that allows the building of more diverse and powerful Web sites and applications.

  2. CSS

    3. Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

  3. Vanilla JavaScript

    4. JavaScript, often abbreviated as JS, is a programming language that conforms to the ECMAScript specification. JavaScript is high-level, often just-in-time compiled, and multi-paradigm. Alongside HTML and CSS, JavaScript is one of the core technologies of the World Wide Web. JavaScript enables interactive web pages and is an essential part of web applications.

  4. PHP

    5. PHP is a popular general-purpose scripting language that is especially suited to web development. PHP code is usually processed on a web server by a PHP interpreter implemented as a module, a daemon or as a Common Gateway Interface (CGI) executable.

  5. MySQL

    6. MySQL is a relational database management system based on SQL â€“ Structured Query Language. The application is used for a wide range of purposes, including data warehousing, e-commerce, and logging applications. The most common use for MySQL however, is for the purpose of a web database.

  6. AJAX

    7. Ajax (short for "Asynchronous JavaScript and XML") is a set of web development techniques using many web technologies on the client side to create asynchronous web applications. With Ajax, web applications can send and retrieve data from a server asynchronously (in the background) without interfering with the display and behavior of the existing page.

  7. XML

    8. Extensible Markup Language (XML) is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable. The design goals of XML emphasize simplicity, generality, and usability across the Internet.

Application architecture

Implementation

We used HTML5 along with PHP for creating the structures of pages using data from the MySQL database, aswell as their dynamic loading realised with AJAX technologies suite.

We used CSS3 to display a simplistic design, but at the same time elegant enchanced by an expressive dynamism realised by JavaScript.

For storing and managing data we used the database administration system named "MySQL" which is a standard for Web applications development.

We also used the AJAX technologies suite for realising some of the dynamic data loads, for example the functionality of the button "Afiseaza mai mult", realised without reloading the page.

For representing the RSS flow we used XML and for the statistics we used the open-source library "ChartJS". When it comes to exporting data, files of type csv or pdf (realised with "jsPDF" library) will be exported.
